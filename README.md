# My First MEAN

My first test project with MongoDB, Express.js, Angular.js, Node.js.

## DB

Run `node createDB` for a create MongoDB base. Script created base `photos`.

## Server

Run `npm start` for a start server. Navigate to `http://localhost:4200/`.
